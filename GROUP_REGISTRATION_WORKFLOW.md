# Group Registration Workflow

## New System Overview

Groups are now registered using a **two-step process**:

1. **Admin creates group slots** with kår assignment and registration codes
2. **Leaders claim groups** using registration codes and set their own group names

## Admin Workflow

### Creating Group Slots

1. Open a project in the admin site
2. Click "Add Group" in the Groups section
3. Enter the **Kår name** (e.g., "Kåren Blå", "Kåren Röd")
4. Click "Create Slot"

The system will:
- Generate a unique 6-character registration code
- Create an "Unclaimed" group slot
- Assign it to the specified kår

### Viewing Group Status

Each group slot shows:
- **Kår**: Which kår the group belongs to
- **Registration Code**: 6-character code (e.g., `A3X9K2`)
- **Status**: 
  - "Waiting for registration" (unclaimed)
  - "✓ Claimed" with group name (claimed)
- **Group Name**: Shows "Unclaimed" until a leader registers

### Sharing Registration Codes

Admin should:
1. Create group slots for each kår
2. Share the registration codes with respective kår leaders
3. Monitor which groups have been claimed

## Leader Workflow

### Registering a Group

1. Open the scanner app
2. Sign in with email/password
3. Select the project
4. Click "Register Your Group" or "+ Register New"
5. Enter the 6-character registration code
6. System verifies:
   - Code is valid
   - Code belongs to this project
   - Code hasn't been claimed yet
7. Enter your group/patrol name (e.g., "Eagles", "Wolves", "Patrol 1")
8. Click "Register Group"

### After Registration

- Group is now claimed and visible in the group selector
- Leader can immediately start scanning for this group
- Group name appears in admin site
- Other leaders can register additional groups using their codes

## Example Scenario

**Admin creates slots:**
- Kår: "Kåren Blå" → Code: `A3X9K2`
- Kår: "Kåren Röd" → Code: `B7Y4M1`
- Kår: "Kåren Grön" → Code: `C2Z8N5`

**Admin shares codes:**
- Sends `A3X9K2` to Kåren Blå leader
- Sends `B7Y4M1` to Kåren Röd leader
- Sends `C2Z8N5` to Kåren Grön leader

**Leaders register:**
- Kåren Blå leader enters `A3X9K2` → Names group "Eagles"
- Kåren Röd leader enters `B7Y4M1` → Names group "Wolves"
- Kåren Grön leader enters `C2Z8N5` → Names group "Bears"

**Result in admin:**
- ✓ Eagles (Kåren Blå) - Claimed
- ✓ Wolves (Kåren Röd) - Claimed
- ✓ Bears (Kåren Grön) - Claimed

## Benefits

### For Admins
- ✅ Control which kår each group belongs to
- ✅ See exactly which groups are registered
- ✅ Know which codes are still unclaimed
- ✅ No need to manually enter group names

### For Leaders
- ✅ Choose their own group/patrol names
- ✅ Simple registration with just a code
- ✅ Immediate access after registration
- ✅ Can register multiple groups if needed

## Database Structure

```typescript
{
  id: "group123",
  projectId: "project456",
  name: "Eagles",              // Set by leader during registration
  kår: "Kåren Blå",           // Set by admin when creating slot
  registrationCode: "A3X9K2",  // Generated by system
  isClaimed: true,             // false until leader registers
  claimedBy: "user789",        // User ID of leader who claimed
  claimedAt: Timestamp,        // When it was claimed
  color: "#3b82f6",           // Random color for UI
  createdAt: Timestamp         // When slot was created
}
```

## Security

- Registration codes are unique per project
- Once claimed, a code cannot be reused
- Only authenticated users can register groups
- Leaders can only see claimed groups in the scanner
